

'''exit''' in computing means to wikt:terminate|terminate a Task (computers)|task or Computer program|program. On many computer operating systems, a Process (computing)|process is stopped by making a system call, called exit. An exit in a Thread (computer science)|multithreading environment means that a thread of execution has stopped running. The operating system reclaims Computer data storage|memory that was used by the process. The process is said to be a ''dead process'' after it terminates.
