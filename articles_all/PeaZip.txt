

'''PeaZip''' is an open source, cross-platform, portable data compression, file manager and encryption software, available for Microsoft Windows and Linux.

PeaZip created and can open the "Pea" archive format, made by the same corporation of the archiver. Peazip is written in Free Pascal, using Lazarus (software)|Lazarus. Peazip is under  the GNU Lesser General Public License, PeaZip is free software.

Peazip can change look, like Opacity (optics)|opacity, colors, and icons.

 Full support formats 
7Z, 7Z-Self-extracting archive|sfx, BZip2/TBZ, GZip/TGZ, LPAQ1/5, PAQ8F/JD/L/O, PEA, QUAD/BALZ/BCM, split (.001), tar (file format)|TAR, WIM, XZ, ZIP (file format)|ZIP

 Just extract or reading formats
ACE (file format)|ACE, ARJ, cabinet (file type)|CAB, Microsoft Compressed HTML Help|CHM, Compound Files|COMPOUND files (MSI, DOC, PPT, XLS...), CPIO, deb (file format)|DEB, ISO image|ISO CD/DVD images, Java archives (JAR (file format)|JAR, EAR (file format)|EAR, WAR (file format)|WAR), LZH, Nullsoft Scriptable Install System|NSIS installers, OpenDocument, PET/PUP (Puppy Linux installers), PAK/PK3/PK4, RAR (file format)|RAR, RPM Package Manager|RPM, WIM, XPI, Z, ZIPX
