

'''Super OS''' (formerly: Super Ubuntu containing various enhancements over Ubuntu (operating system)|Ubuntu. It has the goal of remaining compatible with Ubuntu.

 Features 
Includes an easy way to run executable files and scripts.
Multimedia support: It is prepackaged with support for MP3 files and DVD-Video|video-DVDs.
Internet codecs: It includes an additional web browser (Opera (web browser)|Opera), Adobe Flash Player, and more communication software (aMSN and Skype).
Additional software: It also includes, Wine (software)|Wine, the GUI for Uncomplicated Firewall, and other software installed by default.
