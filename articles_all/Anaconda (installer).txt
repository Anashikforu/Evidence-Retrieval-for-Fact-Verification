
'''Anaconda''' is the installer for Red Hat Linux and Fedora (operating system)|Fedora. It is written in Python (programming language)|Python and C (programming language)|C, with a graphical interface using GTK+|GTK+&nbsp;3 and PyGObject and a text inetrface using python-newt. A Kickstart (Linux)|kickstart file can be used to automatically configure the installation, allowing users to run it without much supervision.

Anaconda is used by Red Hat Enterprise Linux|RHEL, Fedora and a number of other projects, Anaconda offers a text-mode and GUI mode, so users can install on a wide range of systems.

Anacondas are lizard-eating Boidae|boas, and the Caldera OpenLinux|Caldera installation program was called "Lizard", hence this name was chosen.
