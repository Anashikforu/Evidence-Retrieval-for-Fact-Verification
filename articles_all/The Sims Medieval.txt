


'''''The Sims Medieval''''' is a video game developed by The Sims Studio. It is based on ''The Sims'' game series, and was released on March 22, 2011 by Electronic Arts. It is available for Microsoft Windows and Mac OS X. The game is different from the previous versions of ''The Sims'', in the sense that it is set in medieval times. Players can build a kingdom, through a wikt:quest|quest-driven gameplay. The game has received mixed reviews from critics.
