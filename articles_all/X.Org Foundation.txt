The '''X.Org Foundation''' is a group responsible for making the X Window System. The group was created on 22 January 2004.  The '''X.Org Foundation''' is Delaware LLC|a corporation based in Delaware.  It has a non-profit status.

The X.Org Server is the most common implementation of X. It is used on Linux, UNIX, and Unix-like|Unix-like systems. It is the base technology underlying both GNOME and KDE desktops and the Common Desktop Environment|CDE desktop environment
