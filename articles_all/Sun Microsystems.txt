



'''Sun Microsystems, Inc.''' ('''Sun''' for short) was an American company that sold computers, computer components, software, and information technology services. Sun contributed to many important computing technologies, such as Unix, Reduced instruction set computer|RISC processors, thin client computing, and virtualization|virtualized computing. Sun created the Java (programming language)|Java programming language, the Solaris operating system, the Network File System (NFS), and the SPARC instruction set. Sun was founded on February 24, 1982. Sun's headquarters were in Santa Clara, California. In January 2010, it was bought by Oracle Corporation.
