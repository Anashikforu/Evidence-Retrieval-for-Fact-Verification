
'''Baseball-Reference.com''' is a website that lists statistics for every player in Major League Baseball history. The site is often used by large media companies and baseball broadcasters as a source of information.

History
The website came online in April 2000.  It had originally been published in February 2000 as part of the website for the ''Big Bad Baseball Annual''. It was first built as a web version of the Lahman Baseball Database. Now it uses variety of data sources. It has year-by-year team pages, a baseball encyclopedia (the Bullpen, powered by MediaWiki software, using the GFDL),
