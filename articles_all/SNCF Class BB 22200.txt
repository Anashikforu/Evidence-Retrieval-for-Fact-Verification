
The '''BB 22200''' is a class of electric locomotives in service with the French railways SNCF, built by Alstom|Alsthom between 1976-1986. They are a polycurrent version (1500 CV direct current and 25 kV 50&nbsp;Hz AC) of the SNCF Class BB 7200|BB 7200 and SNCF Class BB 15000|BB 15000 classes,The designation "22000" is in fact the sum of 7200 and 15000 with which it shares the exterior appearance. Thanks to this characteristic they can run in all French electrified lines.
