


''' Hattrick ''' is an online football manager game created by  Björn Holmér. It was released on August 20|20 August 1997. The game has an estimated user amount of 1.000.000 ( one million) spread in over 120 countries.

If one registers himself as user he will  receive a team. The user will then play in a competition with other managers. The user has to care for the whole team. This can be done by buying and selling players, training players, creating tactics and by creating line-up|line ups.

 Community 

Hattrick has a 'strong' community. Because of that, not only the forums in-game are active but also those on external sites.
