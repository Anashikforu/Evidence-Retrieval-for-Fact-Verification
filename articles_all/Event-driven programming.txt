In computer programming, '''event-driven programming''' also known as '''event-based programming''' is a programming method in which the Computer program|flow of the program is determined by sensor outputs or user actions (such as computer mouse|mouse clicks, key presses) or by Message (computer science)|messages from other computer program|programs or Thread (computer science)|threads running on the computer. 

Event-driven programs can be written using any of the Programming language|computer programming languages, although the task is easier in languages that provide Abstraction (computer science)|high-level of abstractions to support it. Some integrated development environments provide all assisting methods that automate most tasks required for events handling.
