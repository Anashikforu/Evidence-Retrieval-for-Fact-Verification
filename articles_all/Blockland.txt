

'''''Blockland''''' is a multiplayer computer game made on the Torque Game Engine, where people make things using blocks. Blockland has around 35,500 players. People could build up to 128,000 bricks on every server to keep the game running best. 

In 2009, Lego were talking to Eric "Badspot" Hartman about selling Blockland. It has been talked about by ''The Screen Savers'' Right now, the full game costs $20.
