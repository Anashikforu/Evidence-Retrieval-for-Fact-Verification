'''Multialphabetism''' more than one alphabet being used and can be for different reasons:
Some languages are written using more than one alphabet. For example, Japanese language|Japanese can use kanji, which is based on logograms, or hiragana or katakana, which are Syllabic script|syllabic. Many foreign people know the Roman alphabet, and Romanization of Japanese helps them.
Moldovan language|Moldovan is officially written in the Roman alphabet because it is almost the same as Romanian language|Romanian. In parts of Moldovia, the Cyrillic script is used instead. The written languages are identical but use different scripts.
Multialphabetism can also be the result of speaking many languages and using the respective script for each.
Only certain names are spelled in the original language, and they use special characters.
