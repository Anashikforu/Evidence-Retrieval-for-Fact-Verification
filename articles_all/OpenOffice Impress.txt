

'''OpenOffice Impress''', a part of the OpenOffice office suite package and created by Sun Microsystems, is a presentation program similar to Microsoft PowerPoint.
OpenOffice Impress users can install the Open Clip Art Library, which adds a large amount of images for general presentation and drawing projects. Linux distributions Debian, Gentoo Linux|Gentoo, Mandriva and Ubuntu Linux|Ubuntu have given the ready-to-use ''openclipart'' package for download and install from their online software repositories.
