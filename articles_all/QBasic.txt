

'''QBasic''' is a programming language made by Microsoft which was included with Windows 95 and MS-DOS. QBasic is a more limited version of QuickBasic, released in 1985. While today considered almost wikt:obsolete|obsolete, it is still a popular language for beginners and especially for educational purposes.

 Example 
An example Hello World program in QBasic:


PRINT "Hello, World!"
