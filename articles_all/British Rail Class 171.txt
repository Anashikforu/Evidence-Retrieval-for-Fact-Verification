
The '''Class 171''' "Turbostar" is a type of diesel multiple unit|DMU operated by Southern (train operating company)|Southern. The Class 171 is identical to the British Rail Class 170|Class 170, except for the replacement of the Bergische Stahl Industrie|BSI coupler with a Dellner coupler. This provision was made in order to allow emergency joining with British Rail Class 377|Class 377 DC third-rail electric units.
