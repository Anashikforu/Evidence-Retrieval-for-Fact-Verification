'''''Windows Narrator''''' is a tool for the Eyes|visually Impairment|impaired. The user has four options to customize their  experience.

Purpose

The tool reads text, message boxes, and mimics user typing. The voice, pitch, and speed can all be tailored to the user's preferences.Microsoft Windows comes to Narrator. Official Windows Site.
