'''WebKit''' is a layout engine made to let web browsers show web pages. The WebKit engine has a set of Class (programming)|classes to display web content in GUI|windows, and implements browser features such as following links when clicked by the person using the web browser (the user), managing a back-forward list, and keeping a history of pages recently visited.
WebKit was first made to be used as the layout engine for Safari (web browser)|Safari, and is portable to many other computing platforms.
