On computers using graphical user interface|graphics for an interface, a '''tab''' is a way to switch between different controls (such as in a configuration screen), or documents (such as in a web browser or word processor). Only one tab can be active at a time.



 Use in browsers 
More recently tabs have become a popular feature of most web browsers.  This allows you to have more than one website open in the same window.
